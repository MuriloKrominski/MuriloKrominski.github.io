Otimizado por Murilo Krominski, baseado no Workshop FLAI Dominando Data Science. Medical Cost Data - Análise Exploratória de Dados -
pandas - SweetViz - IPython - ProfileReport - warnings - pandas_profiling - DTale - seaborn - Machine Learning com PyCaret -
pycaret.regression - Random Forest Regressor - Outros modelos - pycaret.classification
Descobre a relaçao dos dados com o custo e faz um modelo para prever se clientes do plano de saúde são potenciais fraudadores, de forma que
seja possível enviar a informação para um departamento para uma investigação mais minuciosa sobre o potencial fraudador, juntamente a
construção uma aplicação web para disponibilizar um acesso ao modelo bastante conveniente para o usuário final.
